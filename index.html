Your Data Stays Yours.

Pasting is designed with local-first privacy. Unlike other clipboard managers, we do not use servers, databases in the cloud, or remote telemetry.

  • 100% Local Storage: Clipboard history is saved locally on your Mac's secure storage.
  • Password Filter: Concealed or auto-generated fields from password managers are automatically ignored.
  • Application Blacklist: Stop Pasting from recording copies inside sensitive applications (e.g. Slack, banking apps).
  • Offline Operation: Pasting runs fully offline. No clipboard data is ever sent to any remote server.

App Store Reviewer Guidelines

Pasting requires **Accessibility API permissions** to perform the simulated keystroke commands needed to automatically paste items back into target applications. Here is how to verify and test the functionality:

  1. Open **System Settings -> Privacy & Security -> Accessibility**.
  2. Click the **`+` (Plus)** button and add **Pasting.app** manually, then toggle the switch **ON**.
  3. Relaunch the app, copy some text, trigger the HUD search panel using the hotkey (`Cmd+Shift+V` by default), and select a row. It will automatically paste back into the active application.

Privacy Policy

Effective Date: June 28, 2026

Pasting operates as a local-only utility application. We value your privacy and are committed to protecting any data associated with your use of the application.

1. Information Collection and Use

Pasting collects and records clipboard data (such as copied text, images, links, and files) solely to provide the clipboard history features within the app. **All collected data is stored locally on your device** in the application container directory.

2. No Data Transmission

Pasting does not operate external servers, database syncs, or network telemetry. No clipboard content, usage data, or diagnostic information is ever uploaded, shared, or transmitted off your local device.

3. Sensitive Application Filtering

The app respects standard password manager transient flags. Copies flagged as auto-generated, transient, or concealed are automatically ignored and never saved to the local database. You can also explicitly blacklist specific applications from being monitored in the app's settings panel.

4. Security of Data

Your clipboard history is saved inside your user profile directory (`~/Library/Application Support/Pasting/`). You can encrypt this data using standard macOS FileVault encryption and protect app entry using the biometric Touch ID lock built into the application.
